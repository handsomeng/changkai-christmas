# 🎄 敞开酒馆圣诞祝福墙

实时互动圣诞祝福系统，支持扫码送祝福、大屏实时显示、3D圣诞树、手势控制。

## ✨ 功能特性

- 📱 手机扫码送祝福
- 🖥️ 大屏实时高亮显示
- 💬 弹幕循环播放
- 🎨 3D圣诞树场景
- 👋 手势识别控制（可选）
- ☁️ 云端实时同步（Supabase）

## 🚀 在线体验

- 大屏端：[敞开圣诞树.html](./敞开圣诞树.html)
- 手机端：[手机端祝福.html](./手机端祝福.html)

## 🛠️ 技术栈

- **前端**: Three.js + WebGL
- **实时通信**: Supabase Realtime
- **手势识别**: MediaPipe Hands
- **部署**: Vercel

扫描大屏左上角二维码 → 填写祝福 → 大屏实时高亮 5 秒 → 转为弹幕滚动
